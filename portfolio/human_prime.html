身份镜像 / Human′

未来网络身份伦理反思

“《身份镜像》是一个交互体验装置，旨在通过AI换脸和H5技术，用采集的观者数据生成观者在未来网络世界中的多个‘身份’，并使这些独立‘身份’在移动端与观者进行对话，以反思大量个人信息被数据化采集并复制之后带来的技术伦理问题，引起并采集观者对未来网络语境下身份伦理问题的反思与讨论。在线上展厅中，观者将以‘Lancy’的身份模拟体验线下展览的场景与交互。”

设计背景

从人类数据化到人工智人

随着身份识别技术的广泛应用，越来越多的生物特征信息正在被数据化采集，并在网络技术的支持下被高速地传输、储存、复制于云端和各种各样的设备当中。而未来网络和人工智能的技术牵引则在人类数据化的基础上催生了“人工智人”的出现。如今，不论是用逝者生前书信和语音创造的聊天机器人“Dadbot”，还是由真人面部声音特征与AI生成的性格行为结合生成的人工智人Neon，都证明了人类增强正在从科幻走向现实；而在未来网络的语境下，随着AI、嵌入式芯片、脑机接口等革命性技术的成熟，人类的感知、计算与表达将被增强，人类将直接成为物联网的终端，甚至脱离肉体的禁锢，意识上传并永生于网络世界中。

问题提出

未来网络身份伦理问题

未来网络技术在给人类创造福祉的同时也带来了潜在的身份伦理问题。哲学家斯拉沃热•齐泽克（Slavoj Žižek）在对《黑客帝国》的解构中认为，删除或改变我们的数字身份可能会把我们变成“非人”；MIT教授Sherry Turkle认为计算机不仅改变了我们做什么，也改变了我们对世界和我们自己的看法。在未来网络语境下，人们在享受网络身份带来的便利和“虚拟化身”提供的自由、快感和想象力补偿的同时，还可能面临身份追踪、身份盗窃的风险，与身份认同、身份分裂、身份迷失的伦理问题。对此，我希望在我的作品中对未来网络身份伦理问题展开思辨：当个人信息越来越多地被数据化采集并复制，甚至在未来网络世界中出现了独立的意识和行为，生成了你的“身份”镜像，你是否会对你的“身份”镜像产生身份认同？你对身份决定因素的认知会是名字、形象、基因还是意识、情感或思想？你是否可能面临存在感或存在价值的消解与重构？

设计概念

从身体到身份，从镜像到对话

面向这些问题，我提出了《身份镜像 / Human′》的交互装置概念。《身份镜像》和《Human′》分别为作品的中英文命名，“身份镜像”顾名思义，“Human′”中的符号角分符号“′”引用了符号的数学含义：在数学中，x′ 通常用于表示与 x 相关或派生自 x 的事物（旋转、平移、反射等），Human′ 意味着现实世界中的 Human 在未来网络世界中的副本。对于未来网络中可能出现的身份伦理问题我无法给出解答，但我希望能够通过交互装置，从身体到身份的复制，从镜像到对话的生成，向观者呈现身份伦理问题，促使观者反思真实世界和虚拟世界中交织的身份与人格问题。

设计说明

系统概述

在《身份镜像》的交互系统中，存在真实的观者与虚拟的镜像两个角色，以及“镜面”装置与观者手机两个交互媒介。观者为交互体验的直接参与者，当观者站在“镜面”装置前，或对“镜面”装置中的二维码进行扫描，都会触发镜像的不同状态转换；镜像为观者在未来网络世界中的映射，它在复制当下现实生活中观者生物特征的同时，呈现未来网络时代人类的特点，并拥有独立的意识；“镜面”装置用于呈现与观者等大的镜像；观者手机作为观者的所有物，通过H5页面的形式呈现身份伦理问题的影响。


《身份镜像》的软件实现系统主要分为数据库、输入、处理和输出4个部分。数据库包括镜像动作数据库和镜像对话数据库。镜像动作数据库内容为视频格式，由Blender制作模型、Mixamo绑定动态、C4D贴图渲染、Ae调色合成；镜像对话数据库内容为文本格式。在输入部分，观者被识别的身份数据主要为人脸数据、身型数据和昵称数据，分别通过摄像头和微信扫码授权采集，并利用人脸/人体识别API挑选出合适角度的面部、判断观者的身型比例。在处理部分，镜像的形象需要通过人脸融合API进行合成，通过Processing生成视频，并将换脸结果接入至H5页面中。在输出部分，“镜面”装置上的画面由投影仪投射于装置上，H5页面经过微信扫码在观者手机中打开。


体验流程

在《身份镜像》的交互尚未开始之时，数据采集已经开始。处于常开状态的摄像头会实时检测并搜集“镜面”装置5米之内所有观者的身体和正、侧面面部数据，以便于后台提前进行换脸预处理。当观者位于“镜面”装置5米之内的中间区域时，“镜面”装置会生成观者的“身体”镜像，以达到吸引观者目光的效果。当观者成功被镜像吸引，以正面朝向“镜面”装置并驻足超过5秒，“镜面”装置会呈现利用AI换脸技术生成的观者在网络世界中的多个“身份”镜像。这些身份拥有观者的面部和未来网络世界中的身体，并会被赋予不同于观者的意识和行为。在“身份”镜像出现10秒后，镜像们会分别邀请观者扫描镜像屏幕上的二维码，以H5页面为载体转移至观者的手机中，通过建立引导式对话启发用户反思身份伦理问题，并搜集用户对身份伦理问题的讨论，同时为观者提供反馈、分享和了解更多的功能。


镜像设计

对于 “身份”镜像的形象设计，我划定了三个关键词——“未来”、“网络”、“换脸”。参考了人工智能Neon、空山基插画中的人物、《机械姬》主角艾娃等人物形象设计，并经过了从现实到抽象的大量设计迭代之后，我总结出了最为合适的设计方式——通过去除性征和现实元素的人体设计来体现未来感，通过添加网格、界面等元素来体现网络世界的特征，通过保留真实人物脸庞、制造现实与虚拟的反差来体现换脸感。


而每个镜像的概念则来源于网络技术发展下的未来人类推测。形象一基于意识上传、肉体消失的未来想象，随着技术的发展，未来的人类能够在网络中存储自己的一切数据甚至意识，人类的生命不再依赖于肉体这一短暂载体，而在网络中永恒；形象二基于人体联网、人类数据化的未来推测。植入式芯片和柔性屏的成熟能够让人类也成为万物互联的终端，人类器官的一切变化都将可视化于网络之中，皮肤也将直接成为交互的媒介；形象三基于性征隐藏、性取向自由的未来愿景。当肉体不再是人类生存于世上的唯一载体，生育不再是人类繁衍的唯一方式，那么性征和性取向也不再受社会伦理观念的禁锢，真正达到自由。


对话设计

H5页面中对话的语义模型的定义场景分为镜像主动设问和观者主动探索两种情况。镜像主动设问的意图主要为思辨，包括思辨身份同一性、思辨身份决定因素、思辨人性；观者主动了解的意图主要为探索和创造。探索包括了解镜像、了解作者、了解作品，创造为创建群聊。对于镜像主动设问的意图，每种意图需要设置1-3种不同的引导式询问话术，并根据观者不同答复可能性设置1-5种追问话术。对于观者主动探索的意图，每种意图需要设置1-5种观者说法可能性、1-2种镜像的默认答复及承接至镜像主动设问的意图串联逻辑。


以镜像主动设问的思辨身份同一性的意图为例。镜像会通过询问“你觉得我们是同一个人吗？”或“我是你吗？”来获得观者对身份同一性问题的答复。不论观者的答复倾向与肯定、否定或模糊，镜像都会通过追问来尝试动摇观者的固有印象：“我们明明拥有同样的眼睛，同样的头发。”“可是我们的性格完全不一样。”经过追问，观者因追问而产生的答复倾向变化会被后台记录，用控制变量的方式探索观者对身份决定因素的看法。


界面设计


展示设计

参考了海瑟·德维-哈格伯格、Es Devlin、Didier Faustino在作品中使用的复制手法，为营造镜像复制的震撼感和沉浸的体验观感，《身份镜像》的线下装置设计由6块交错摆放的展架组成，展厅整体氛围选取封闭式的暗空间。展架足够投映等人高的镜像，投影材料选取硫酸纸。由于疫情的原因，实体装置无法于毕设结束前在线下展出，因此我选择就近选择场地搭建装置原型（图4-9b），并将线下观展的场景和核心交互内容尽可能保留至线上展览中。


《身份镜像》的线上展览于清华大学美术学院线上毕业作品展中展出，在线上展览里，观者将以“Lancy”的身份体验线下观展的模拟场景和核心交互内容。划开作品信息墙，首先将看到位于中远景的观者“Lancy”和“镜面”装置中的“身份”镜像“Lancy′”们（图4-10a），接着将在近景“Lancy”的手机中与“Lancy′”进行对话（图4-10b），最后是超近景中的写有设计背景与概念的半透明信息墙（图4-10c），在右下角作品集按钮中，观者可以查看更多设计说明。
